# Paradise Nursery - e-plantShopping

An e-commerce front-end application for buying houseplants.  
Built with React, Redux Toolkit, and React Router.

## Features
- Landing page with company intro and background image
- Product listing page with plants grouped by categories (Air Purifying, Aromatic, etc.)
- Shopping cart with quantity controls, delete option, and total cost
- Persistent cart state using Redux

## Technologies
- React 18
- Redux Toolkit
- React Router DOM
- Vite

## How to run locally
```bash
npm install
npm run dev
